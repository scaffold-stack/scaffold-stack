# stacksdapp-scaffold

The project initialization and UI engine for `stacksdapp`.

This crate contains the frontend templates and scaffolding logic for the `stacksdapp new` command. It bundles a high-performance Next.js starter kit pre-configured with Stacks wallet providers.

### Features
* **Rapid Prototyping:** Bootstraps full-stack projects in seconds.
* **UI Components:** Includes pre-built React components for contract interaction.
* **Modern Stack:** Pre-configured with Tailwind CSS and specialized Stacks hooks.
