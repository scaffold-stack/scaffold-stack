# stacksdapp-process-supervisor

Reliable process management for Stacks development environments.

A supervisor tree used to orchestrate background services. It manages the lifecycle of local Bitcoin/Stacks nodes and development servers, ensuring a stable environment for dapp development.

### Features
* **Lifecycle Management:** Handles graceful starts, stops, and restarts.
* **Log Aggregation:** Streamlines output from multiple sub-processes.
* **Stability:** Ensures local devnet nodes remain healthy during testing.
